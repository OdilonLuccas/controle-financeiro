# Controle Financeiro Pessoal

Projeto simples de controle financeiro desenvolvido em Excel com objetivo de organizar receitas, despesas e gastos em cartões de crédito.

## Objetivo

Criar uma ferramenta simples para controle financeiro doméstico que possa evoluir para:

- Dashboard no Power BI
- Integração automática com extratos bancários
- Possível aplicação futura

## Estrutura

A planilha possui:

- Resumo mensal
- Controle de receitas
- Controle de despesas
- Reservas e investimentos
- Controle de cartões de crédito

## Tecnologias utilizadas

- Microsoft Excel
- (Futuro) Power BI
- (Futuro) Automação de dados financeiros
